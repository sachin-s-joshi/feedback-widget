üìö Implementation Guide

1. NPM Installation

npm install @yourorg/feedback-widget

2. Basic Usage

import feedbackWidget from '@yourorg/feedback-widget'; // Initialize with configuration feedbackWidget.init({ id: 'my-feedback-widget', version: '1.0.0', position: 'bottom-right', triggers: [ { type: 'time', conditions: { timeDelay: 5000 } } ], fields: [ { type: 'rating', label: 'Rate your experience', required: true } ], dataLayer: { enabled: true, eventName: 'feedback_submitted' } });

3. CDN Usage

<script src="https://cdn.jsdelivr.net/npm/@yourorg/feedback-widget/dist/feedback-widget.min.js"></script> <script> window.feedbackWidget.create('demo-widget', { triggers: [{ type: 'manual' }] }); </script>

4. Event Handling

// Listen for feedback submissions feedbackWidget.on('feedbackSubmitted', (data) => { console.log('Feedback received:', data); }); // Manually trigger widget feedbackWidget.show('my-widget-id');
